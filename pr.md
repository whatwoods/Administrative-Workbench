Administrative Workbench

核心功能需求
1. Todo List管理
分类：维修事项/工程项目/日常工作

支持按项目、紧急程度筛选

增删改查、拖拽排序、状态标记

2. 时间与气候信息面板
当前天气：温度、天气状况、日落时间

未来5天天气预报

当前节气信息：节气名称、日期范围、园区亮灯时间

下个节气倒计时（悬停展开完整24节气列表）

3. 智能便签组件
支持文本输入和绘图

多便签管理

自动保存，支持版本历史

4. 费用统计卡片
手动录入界面（快速表单+批量导入）

分类统计：办公维修/工程维修/水/电/燃气费

可视化图表：月度趋势、报销进度、剩余额度

5. 快速导航栏
固定侧边栏（可折叠）

分组链接：行政系统/财务/员工服务/外部链接

支持自定义和搜索

云端同步：用户自定义导航项同步

6. AI智能助手
集成LLM API

功能：

智能问答（基于页面数据）

智能创建待办事项

智能录入费用

智能生成文档到便签

云端部署数据逻辑
数据存储策略
本地缓存：临时数据、界面偏好

云端主存储：所有业务数据（Todo、费用、便签）

用户认证：账号系统支持多设备登录

数据同步机制
实时自动同步

离线支持（网络恢复后自动同步）

冲突解决（基于时间戳或用户选择）

权限管理
角色区分：管理员、普通用户

数据隔离：用户只能访问自己的数据（管理员除外）

操作日志：关键操作记录

基础要求
响应式设计（桌面/平板/移动端）

数据备份与恢复机制

性能优化（懒加载、图片压缩）
支持docker一键部署
